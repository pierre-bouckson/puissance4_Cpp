## Compiler

Utilisez gcc ou n'importe quel compilateur C++ :

``` bash
gcc puissance4.cpp -o puissance4
./puissance4
```

## Jouer

-   `q` : se déplacer vers la gauche\
-   `d` : se déplacer vers la droite\
-   `s` : valider le choix

Quand il y a un gagnant, utilisez **CTRL+C** pour quitter puis relancez
`./puissance4`.
